It's been a busy weekend!  We released a first preview of milestone 1
on winforms under a new UI/Render plugin combination called "Third
Life".  You can download a zip and run OpenViewer.exe from the link
listed on http://code.google.com/p/thirdlife/.  Please note that this
is Windows only for now =)  Here's a list of some of the features we
have:

- Completely modular UI
- Embedded rendering of metaverse
- Ability to send out chat

Sometime this week Jen and I would like to talk with OpenViewer about
expanding the interfaces on the rendering module.  I noticed two
limitations this weekend:

- I'd like to expand the interface to support more input into the
rendering module from the UI ( both keyboard and mouse )
- Also, there isn't a way to pass in resizing events to the renderer
(in cases where usercontrols/windows resize).   This would be very
helpful to us, since we now have a resizable embedded renderer, but no
way to tell ogre3d to resize when our window does ;)

Anyhow, mail me back your thoughts! Thanks

- Daedius
