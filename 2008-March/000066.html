Hello Openviewers.  I wanted to write you all an email on the state of
all things GTK related.  First of all, we've had alot of great
progress over the last weekend, and I thank the many people for
putting together the ideas and contributions to help get GTK UI
started right!  To name a few of the things we now have:

- The foundations of a UserSettings
- A solid MVC pattern behind the GTK UI ( code manageability )
- Login sleekness (dropdown boxes of recent grids, etc.)

I want to organize and pave the way for what i'm envisioning as the
first milestone of GTK UI:
- Login Screen
- Chat Screen ( Local Chat/IM/Group Chat)
- OpenViewer Capabilities Management
- UserSettings
- Settings Screen
- Bonus Feature: 3D Integration ( if its possible, if not, having an
external window will be fine ) =o

Anyhow, there is wiki page for GTK UI Milestone 1
(http://www.openviewer.org/index.php/GTKUIMilestone1) I will keep open
for a day or so.  Please help by contributing your ideas and thoughts
to Milestone 1 here or on the wiki. Have a great week!

- Daedius
