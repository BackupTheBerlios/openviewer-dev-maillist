This patch includes changes for adding an nunit testing project for
the SL Protocol.  Right now I just test login but I think its a great
first start.  You will need the nunit.framework.dll binary from nunit
website ( get the .net 2.0 version )

Compile as normal.

to run tests

cd bin
nunit-console2 OpenViewer.Protocol.SL.Tests.dll

I created a test account named Open Viewer to run our tests with.

Enjoy!
