Thanks for the link Nink,

I think if openviewer ever became an embedded plugin into a firefox
browser and was the focus of connecting into a single metaverse, you'd
be correct.   Honestly though, open viewer as it looks like now with
GTK UI is going to be used to connect into a variety of different
types of grids.  Each with specific settings and saved or not saved
passwords.  While using a URL as a shortcut into the whole process is
beneficial, a user still might need more configuration over how they
are connecting and how they interact with features the grid may offer.
 I guess i'm just not sure why you think menus and settings can be
tossed out the door ;)  Thanks for the vid though.
