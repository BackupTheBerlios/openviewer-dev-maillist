GTK UI Milestone 1 has been formalized.  Thank you to everyone for
their ideas!  Let me know if you really have a burning desire to take
ownership of one of these ideas.  Otherwise, I'll be handing out tasks
or working on them myself  =)  Happy hacking everyone!

http://www.openviewer.org/index.php/GTKUIMilestone1

    * Get a good looking logo for OV
    * Provide a setting to override splash screen
    * Get a good looking about box in its own view
    * Complete the connection settings to make like pidgin
    * Get local chat working
    * Drop glade, glade has too many bugs, and in the end, we'll need
our views to be specified in GTK for dockability and proper seperation
    * Be able to safely store passwords in user settings
    * Move user settings downward in UI project
    * Get an address bar working with simple "mtvs://daedius
moskvitch at osgrid.org", should prompt for password if we don't have one
saved and connect
    * Keyboard shortcuts
    * Icons
    * Documentation

Note, if your idea didn't make it into this round, we'll likely be
getting to it next milestone.  The goal of this round is to make
something reasonable and achievable!  Let's do it!

- Daedius
